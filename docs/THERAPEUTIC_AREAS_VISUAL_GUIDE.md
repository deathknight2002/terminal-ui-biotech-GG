### Area Card States

**Default (Inactive)**

```
┌─────────────────────────┐
│ Duchenne Muscular       │ ← Border: rgba(148, 163, 184, 0.15)
│ Dystrophy               │   Background: rgba(11, 15, 21, 0.6)

```

**Hover**
```
┌─────────────────────────┐ ← Transform: translateX(4px)
│ Duchenne Muscular       │   Border: rgba(125, 249, 255, 0.3)
│ Dystrophy               │   Glow: box-shadow cyan
│ Description...          │
└─────────────────────────┘
     ◀── 4px shift
```

**Active (Selected)**
```
┌─────────────────────────┐
│ Duchenne Muscular       │ ← Border: rgba(125, 249, 255, 0.5)
│ Dystrophy               │   Background: rgba(125, 249, 255, 0.05)
│ Description...          │   Strong glow
└─────────────────────────┘
```

### Chart Interactions

**Hover on Data Point**
```
     Unmet Need
         ◢
    ╱────●────╲     ← Point enlarges + glow
  ╱──────┼──────╲     Tooltip appears:
╱────────┼────────╲   ┌──────────────────┐
                      │ DMD               │
Polygon glows         │ Unmet Need: 9.5   │
Stroke width: 3px     │ (Extremely High)  │
                      └──────────────────┘
```

**Legend Toggle**
```
LEGEND
▬▬▬ DMD (active)      ← Click to toggle
▬▬  Cardiology (off)  ← Grayed out, polygon hidden
▬▬▬ IBD (active)
```

## Accessibility Features

- **WCAG AAA Contrast**: 7:1+ on all text
- **Keyboard Navigation**: Tab through cards, Enter to toggle
- **Screen Readers**: ARIA labels on all interactive elements
- **Focus Indicators**: Visible cyan outline on focus
- **Touch Targets**: 44px+ minimum for mobile
